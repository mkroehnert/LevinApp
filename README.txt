
ABOUT
-----

Levin is about playing Adobe shockwave-flash files without opening a browser.
It runs on Mac OS X and may be ported to GNUstep (Linux and Windows) if a
WebKit port/replacement is available which allows loading plugins into the
webpage.



FEATURES
--------

+ Open Source.
+ Play swf files.
+ Native Cocoa interface.
+ Add/remove directories to scan for swf files.
+ Store added directories permanently.



REQUIREMENTS
------------

Computer running Mac OS X with installed flash plugin.
This should be any standard Apple computer with Mac OS X installed.
